| Classification | Meaning |

| ApplyDirectly | standard applies to Fabric v10 as written |
| AdaptForFabric | intent applies, implementation must change for Fabric |

## 3. Required Challenge Checks

Before finalizing Bob alignment, explicitly answer:

- Does Bob's PowerBI view rule still apply when the semantic model is Direct Lake?
- Is v9 `vw_table_dictionary` sufficient as an Enterprise Dictionary Adapter, or does Bob require a physical TableDictionary rowset in EnterpriseData?
- Should schemas use Bob DOCX suffixes (`WRK`, `ENH`, `DW`) or Bob email's Pascal Case convention without suffixes?
- Which source-like objects are truly Bronze vs significant enhancement that must move to Silver/ENH?
- Which ADW-specific performance rules are not applicable to Fabric Warehouse?
- Which security controls belong in Fabric workspace/item/semantic model versus SQL endpoint/schema grants?
